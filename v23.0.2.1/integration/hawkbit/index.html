    Device onboarding with Hawkbit

    Eclipse hawkBit is a domain-independent back-end framework designed for deploying software updates to a variety of devices. It can handle updates for constrained edge devices as well as more powerful controllers and gateways connected to IP-based networking infrastructure. It facilitates easy and flexible rollout management, enabling updates for large numbers of devices in separate groups.

    Overview

    To have a device connect and authenticate with Hawkbit, it needs Hawkbit generated key. This adds additional layer of device onboarding step into your existing device onboarding process. Scurid's integration with Hawkbit removes this additional step and makes it seamless for the device to connect and authenticate with Hawkbit.

    Key highlights

    • Removes the need for custom scripts to distribute Hawkbit generated keys to the devices.
    • Reduce the time to onboard new devices.
    • Low touch, existing Scurid onboarding process can be used to onboard devices with Hawkbit.

    Connect your Hawkbit instance with Scurid Server

    1. Launch Scurid App and navigate to Settings > Integrations.
    2. Click on Hawkbit and enter the following details:
      • Hawkbit URL: Enter the URL of your Hawkbit instance.
      • Hawkbit Username: Enter the username of your Hawkbit instance.
      • Hawkbit Password: Enter the password of
    3. Click on Save.

    Automate device onboarding with Hawkbit

    1. Navigate to Agents click on Approved tab.
    2. Select the Agent, which corresponds to your device, and click on Integrations.
    3. Click on the slider next to Hawkbit to enable the integration.
    4. This triggers following automated steps:
      • Scurid server will request a new API key for the device from the Hawkbit server.
      • Download this key from Hawkbit server.
      • Push the API key to the device running Scurid Edge agent.
    5. And that's it! Your device now has required API key to authenticate with Hawkbit.
    6. This new device can also be viewed in the hawkBit UI.

    Accessing the API key on the device

    1. Connect to your device running Scurid Edge agent e.g. using SSH.
    2. Navigate to the store/ directory. By default, it is created in the same folder where Scurid Edge agent is running. Check the config.yaml file for the API key.
